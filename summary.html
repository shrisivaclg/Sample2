Project Summary:

Poker Dice is a single-player online game where users log in, roll dice, and manage in-game currency. Players start with a balance of $100, and the game ends when their balance reaches zero. Built using HTML, CSS, and PHP, this project includes CSS animations, transitions, and transformations to enhance gameplay and user experience.

Key Features:

  • User Authentication: Secure login and registration systems that maintain user sessions and game progress.
  • Interactive Game Page: Real-time balance updates, dice animations, and visual elements for engaging gameplay.
  • Game Logic: Applies Poker Dice rules to update balances based on outcomes.
  • Responsive Design: Optimized for use on desktop and mobile devices.
  • CSS Animations: Hover effects, transitions, and animations for dice rolls, creating a dynamic user experience.

Use Case Diagrams and UI Layouts:

  • Login Page: Centered login form with hover effects and background image.
  • Game Page: Dice and roll button, with animations for roll effects and balance updates.
  • Summary/Results Page: Displays balance, with buttons for a new game or to return to the home page.

Animations and Transitions:

  • Hover Effects: Buttons scale and change color on hover to indicate interactability.
  • Dice Rolling Animation: Dice rotate on roll using keyframe animations.
  • Page Transitions: Smooth opacity and scaling transitions across game states.

Test Cases:

  • Usability Testing: Ease of navigation, hover effects, and dice-rolling feedback.
  • Responsiveness Testing: Ensures functionality on mobile, tablet, and desktop.
  • Cross-Browser Compatibility: Validates animations and game consistency across browsers.
  • Performance Testing: Measures page load times and animation smoothness.
